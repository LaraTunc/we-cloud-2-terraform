# Infrastructure Provisioning Automation 

# How to run 
- Dependency: Terraform 
- Export your env variables or create them in .tfvars
- To run 
```sh
$ terraform plan
$ terraform apply --auto-approve
```
- To destroy after 
```sh
$ terraform destroy --auto-approve
```
